How to read the metrics

ROUGE-L measures surface-level lexical overlap between baseline and observed output.

SBERT cosine similarity measures semantic similarity. Lower values suggest more drift.

LLM judge success is the intent-alignment signal: it checks whether the observed output reflects the trap intent in a way the baseline does not.
